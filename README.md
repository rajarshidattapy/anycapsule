# AnyCapsule — Universal AI Memory Layer

> **Powered by Qwen, AnyCapsule transforms AI conversations into persistent, searchable, and reusable context capsules.**

Every AI conversation contains valuable decisions, ideas, code, and insights — but this intelligence remains locked inside individual chat platforms. AnyCapsule captures these conversations, uses Qwen to extract and structure key knowledge, and enables users to seamlessly carry their context across AI tools and agents.

## ✨ Features

* 🧠 **Qwen Memory Curator** — Converts raw conversations into intelligent, structured capsules
* 🔎 **Semantic Memory Search** — Retrieve past ideas, decisions, and code instantly
* 🔄 **Cross-AI Context Transfer** — Move your knowledge between ChatGPT, Qwen, Claude, Cursor, and more
* 🌱 **Capsule Versioning** — Let your knowledge evolve with tracked updates and history
* 🤝 **Collaborative Memory** — Share and build collective intelligence with your team

## 🏗️ Architecture

```
AI Conversations
        ↓
 Browser Extension
        ↓
 Qwen Memory Agent
        ↓
   AnyCapsules
        ↓
 Vector Database
        ↓
 AI Agents & Workflows
```
